# pi-raindrop

List, create, update, and manage Raindrop.io bookmarks and tags from Pi.

## Install

```bash
pi install https://github.com/goofansu/pi-raindrop
```

After installation, Pi registers the `raindrop` tool.

## Configuration

Set `RAINDROP_API_KEY` to a Raindrop.io API key or test token used as a Bearer token.
